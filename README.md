# <img src="https://upload.wikimedia.org/wikipedia/commons/1/13/Arch_Linux_%22Crystal%22_icon.svg" height="32" /> arch-clean
> Clean up your Arch Linux system

Bash script that cleans up your Arch computer to free up space, including orphaned packages (including AUR), cache, and temporary files.
Works on Arch forks - EndeavourOS, Manjaro, etc.

## Requirements

You must have Paru installed.
